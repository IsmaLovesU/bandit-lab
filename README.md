# Lab #1 – Web

Este laboratorio tiene como objetivo practicar el uso de la línea de comandos en Linux y el flujo de trabajo con Git y GitHub mediante la resolución de los retos de **OverTheWire Bandit**.

## Objetivos
- Practicar comandos básicos e intermedios de Linux.
- Comprender el manejo de archivos, directorios y permisos.
- Utilizar pipes, redirecciones y comandos de búsqueda.
- Aplicar un flujo de trabajo con Git basado en commits atómicos.
- Documentar procesos técnicos de forma clara y ordenada.

## Herramientas utilizadas
- Terminal Linux
- SSH
- Git y GitHub
- OverTheWire Bandit

## Alcance del laboratorio
Se resolvieron y documentaron los niveles de **Bandit desde el nivel 0 hasta el nivel 10**, incluyendo los comandos utilizados y una breve explicación de cada solución.

## Estructura del repositorio
- `README.md`: descripción general del laboratorio.
- `bandit-report.md`: documentación detallada de cada nivel resuelto.
